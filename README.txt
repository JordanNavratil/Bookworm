Bookstore Site

By Jordan Navratil, October 20, 2015

A bookstore website built using Drupal

Setup:
- Clone repository: https://github.com/JordanNavratil/Bookworm.git
- Import the database found in sites/db-backup to MySQL
- Database Credentials:
    username: bookstore
    password: bookstore
- Start Apache and PHP servers
- Browse to localhost:8888
- Site Admin Login:
    username: jordan
    password: jordan

Technologies Used:
Drupal 7.40
Modules: CTools, Features, Strongarm, Views
